IoT Project

KnockSense

Smart Knock Alert System for Deaf & Mute Users

An assistive IoT device that helps deaf and mute individuals detect door knocks using vibration feedback, LCD messages, and buzzer alerts with pattern recognition for different visitors.

Project Description

KnockSense is an assistive IoT project designed to help deaf and mute individuals detect door knocks using vibration feedback, LCD messages, and a buzzer alert. It classifies different knock counts into meaningful alerts, ensuring safety, accessibility, and inclusiveness.

Project Overview

This system gives multi-sensory alerts — LCD text, buzzer, and vibration — so users who cannot hear knocks can still identify visitors. Perfect for deaf, mute, and elderly individuals. The system uses ESP8266 for future mobile alert integration and is designed to be low-cost and beginner-friendly.

Key Features

Pattern Recognition

Detects knock patterns: 3 knocks → Husband, 4 knocks → Son, 5 knocks → Warning/Unknown

Vibration Feedback

Uses vibration motor to provide tactile alerts for deaf users

LCD Display

Shows clear visitor information on 16×2 LCD screen

Buzzer Alert

Additional auditory feedback for users with partial hearing

WiFi Ready

ESP8266 enables future mobile alert integration

Accessibility Focused

Designed specifically for deaf, mute, and elderly users
