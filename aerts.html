AERTS

Autonomous Edge-AI Real-Time Surveillance & Decision System

A real-time edge device that captures live video/audio, runs on-device ML inference with <50 ms latency, and provides cloud-assisted analytics with a full-stack dashboard for monitoring and alerts.

Core Idea

AERTS is a real-time edge device that:

  1. Captures live video/audio streams
  2. Runs on-device ML inference (object detection, anomaly prediction)
  3. Makes decisions with <50 ms latency
  4. Coordinates with cloud microservices for heavy analytics
  5. Provides a full-stack dashboard with live monitoring, model performance tracking, and device-state telemetry

Technologies Used

Edge AI IoT ML/DL TensorRT ONNX Real-time OS WebRTC/RTSP Kafka/Redis MQTT React.js Node.js Microservices

Technical Architecture

System Architecture Diagram

Edge Device → Cloud Microservices → Dashboard

Key Technical Components

  • Model Compression & Quantization: TensorRT, ONNX runtimes for optimized inference
  • Real-time OS Constraints: Deadlines, scheduling, memory ceilings
  • Low-latency Streaming: RTSP, WebRTC protocols
  • Cloud Inference Fallback: For non-critical frames
  • Event-driven Microservices: Scalable cloud architecture
  • Message Queues: Kafka, Redis Streams, MQTT at the edge
  • Dashboard UI: Analytics, heatmaps, logs with real-time updates
